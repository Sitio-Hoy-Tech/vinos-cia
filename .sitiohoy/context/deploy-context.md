# Deploy Context - Vinos & Cia

## Antes de deploy

- `npm run build`
- `npm run sitiohoy:qa`
- `SITE_URL=http://localhost:3000 npm run sitiohoy:visual-audit`
- `npm run sitiohoy:qa-report`
- Revisar variables en Vercel.
- Confirmar dominio y SSL.
- Probar MercadoPago en produccion y webhook final.

- Verificar Umami pageviews/eventos.
